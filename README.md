# Sample HTML & CSS Project

This project demonstrates a simple website using HTML and CSS. It includes basic pages such as Home, About, Contact, and Login, along with a sample JavaScript file for interactivity.

## Login Page

The project now includes a `login.html` page with a styled login form. The form validates email and password fields using JavaScript (`script.js`). Validation ensures the email is in a valid format and the password is at least 6 characters. On successful validation, a success message is shown.
